## Установка
```bash
git clone git@github.com:azbykov/Bbot.git bbot
cd bbot
npm install
```

## Настройка
Указать в  файле `config/default,js`

```js
bot: {
	auth: {
		login: LOGIN,
		password: PASSWORD
	},
	mail: {
		service: MAIL_SERVER	// например 'mail.ru',
		auth: {
			user: LOGIN,
			pass: PASSWORD
		}
	},
}
```

Настройка cron

```cron
 # добавить в PATH домашний каталог пользователя
PATH= PATH_TO_BBOT_DIRECTORY

```


## Запуск
``
node bot/daily-15
``


## CRON
